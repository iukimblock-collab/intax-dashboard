## 주요 기능

### 인증 · 권한
- 로그인 / 계정 생성 (localStorage 기반)
- **대표** · **직원** 이중 역할 시스템
  - 대표: 전체 메뉴 + 편집 권한
  - 직원: 조회 전용 (수정 버튼 비활성화)

### 대시보드
- **핵심 지표 KPI** 4열 카드
  - 총 기장 거래처 수 / 총 신고대리 거래처 수 / 기장료 합계 / 재직 직원 수
  - 월별 · 분기 · 연간 기간 탭 전환
  - KPI 숫자 클릭 → 거래처 현황 필터 자동 연동
- **기장료 구간 분포 차트** (기장 거래처 전용)
  - 구간: 10만 이하 / 10~20만 / 20~30만 / 30만 초과
  - 막대 클릭 → 거래처 현황으로 이동 + 해당 구간 필터 자동 적용
- **최근 등록 거래처** 목록 (상위 5건)
- **직원 현황** 요약 (담당 수 · 기여 기장료)
- **시스템 카드** (대표 전용)
  - 홈택스 동기화 상태 · 데이터 백업 / 가져오기 / 초기화

### 거래처 현황
- 거래처 등록 · 수정 · 삭제 (드로어 패널)
- **필터바** : 구분 · 유형 · 업체명 · 사업자번호 · 상태 · 기장료 구간 · 담당자(미지정 포함)
- **기장료 열** 클릭 → 해당 구간 필터 즉시 적용 (링크 스타일)
- 열 정렬: 구분 · 업체명 · 기장료 · 담당직원
- CSV 내보내기 (현재 필터 · 정렬 적용 기준)
- 홈택스 연동: JSON 파일 가져오기 · 사업자 상태 검증 (NTS API)
- 모바일: 상태 검증 · 거래처 등록 버튼 topbar 하단 별도 행으로 분리

### 직원 현황
- 직원별 카드
  - 담당 거래처 (법인 / 개인) · 기여 기장료
  - **연차 현황**: 발생 · 사용 · 잔여 3분할 표시
- **계정 관리 통합 테이블**: 직원 수정 + 계정 수정 버튼 통합
  - 이름 · 권한 · 아이디 · 재직 상태 · 입사일 · 사용 연차 · 아바타 색상
- 직원별 기여 기장료 · 담당 수 Bar 차트

### 연차 자동 계산 (근로기준법 기반)
| 근속 기간 | 발생 연차 |
|-----------|-----------|
| 1년 미만 | 완성 개월 수 × 1일 (최대 11일) |
| 1년 이상 | 15일 기본 + 3년차부터 2년마다 +1일 (최대 25일) |

- 사용 연차는 **계정 수정** 모달에서 입력
- 잔여 연차 = 발생 - 사용 (색상: 초록 / 노랑 / 빨강)

### 기장료 관리 (대표 전용)
- 기장료 현황 조회

### 안내문 발송
- 거래처 대상 안내문 작성 · 발송
